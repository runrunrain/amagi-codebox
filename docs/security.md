# Security Policy

## Supported Versions

| Version | Supported |
|---------|----------|
| 1.0.x   | Yes      |
| < 1.0   | No       |

## Reporting a Vulnerability

If you discover a security vulnerability in Amagi CodeBox, please report it responsibly:

1. **Do NOT open a public issue.**
2. Send an email to **maorun@live.cn** with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)
3. You will receive an acknowledgment within 48 hours.
4. A fix will be prioritized and released as a patch version.

Thank you for helping keep Amagi CodeBox secure.
